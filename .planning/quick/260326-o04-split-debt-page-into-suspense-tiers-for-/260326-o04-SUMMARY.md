### Task 2 — Refactored deudas/page.tsx

**Tier 1 (renders instantly):**
- `await connection()` + `await searchParams` + `await getPreferredCurrency()`
- MobilePageHeader + MonthSelector
- Desktop h1 + desktop MonthSelector + Planificador button

**Tier 2 (streams in via Suspense):**
- `DebtOverviewSection` async Server Component fetches `getDebtOverview`, `getEstimatedIncome`, `getExchangeRate` in parallel

- All data derivation (creditCards, loans, secondaryCurrencies, salaryBreakdown) inside DebtOverviewSection
- Suspense fallback stacks all four skeleton components in a `space-y-6` div
