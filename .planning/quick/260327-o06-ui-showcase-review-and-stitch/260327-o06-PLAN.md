# Quick Task Plan — UI showcase review and Stitch handoff

## Goal

Review the desktop and mobile showcase files as design artifacts, identify why the product still feels broad and directionless, and create a smaller Stitch-ready package that is easier to use for redesign exploration.

## Scope

- Audit `ui-showcases/desktop-showcase.html`
- Audit `ui-showcases/mobile-showcase.html`
- Review the screenshot set for usefulness as design input
- Produce a written review with concrete product/UI changes
- Produce a copy-paste prompt for Stitch
- Generate a compact visual board from the most important screens

## Constraints

- Do not modify live app code
- Keep the artifact self-contained under `ui-showcases/`
- Prefer curation over dumping every screen equally
- Preserve the product direction already established in `ship-now/`

## Deliverables

- Written review
- Stitch brief / prompt
- Static preview page for the new package
- Curated image board(s) and per-screen paired references
