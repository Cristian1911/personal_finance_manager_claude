    **1. Create `webapp/src/components/dashboard/status-headline.tsx`** (Server Component, no "use client"):

    Props interface:
    ```typescript
    interface StatusHeadlineProps {
      allocationData: AllocationData | null;
    }
    ```

    Logic (per D-02):
    - If `allocationData` is null: render prompt-to-action "Importa tu extracto para ver tu estado" in `text-muted-foreground`, no colored dot
    - Compute `spentPercent = ((allocationData.needs.amount + allocationData.wants.amount) / allocationData.income) * 100`
    - If `spentPercent < 90`: green dot (`bg-z-income`), copy: "Vas bien este mes — gastos dentro del presupuesto"
    - If `spentPercent >= 90 && spentPercent <= 100`: yellow dot (`bg-z-alert`), copy: "Cerca del limite — queda poco margen este mes"  (use accent: limite)
    - If `spentPercent > 100`: red dot (`bg-z-debt`), compute overAmount = `(allocationData.needs.amount + allocationData.wants.amount) - allocationData.income`, copy: "Te pasaste del presupuesto — ${formatCurrency(overAmount, allocationData.currency)} por encima"

    Render structure:
    ```tsx
    <div className="flex items-center gap-2 mt-3">
      {dotColor && (
        <span
          className={`inline-block h-2 w-2 rounded-full shrink-0 ${dotColor}`}
          aria-label={ariaLabel}  // e.g. "Estado: dentro del presupuesto"
        />
      )}
      <p className="text-[15px] text-muted-foreground leading-relaxed">{copy}</p>
    </div>
    ```

    Typography: 15px / weight 400 per UI-SPEC body role. Use `text-[15px]` since it matches the body token exactly.
